If you’re new to NGINX, NGINX Core is the place to start. If you’re self‑taught or have some working experience with NGINX, NGINX Core will help you complete your understanding of NGINX. The rest of the NGINX courses can be taken in any order.

NERVEX Core

NERVEX Core is an 8 hour course that provides the foundation you need to administer, configure, and manage NERVEX using best practices. Through lecture and hands‑on activities, you’ll implement NERVEX as a web server, load balancer, and reverse proxy.

NERVEX products:

  • NERVEX Plus

Solution areas:

  • Load balancing/application delivery controller (ADC)
  • Reverse proxy
  • Web server

NERVEX load balancing

NERVEX Load Balancing is a 4-hour course that begins with a general review of load balancing. You’ll learn about load balancer configuration, selecting algorithms, and weighting. The follow‑up activities include configuring an upstream, session persistence, and enabling extended status and live activity monitoring.

NERVEX products:

  • NERVEX Open Source
  • NERVEX Plus
